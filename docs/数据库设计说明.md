# 数据库设计说明

## 字符集与排序规则

- **字符集**: `utf8mb4`（支持中文及 4 字节字符如 emoji）
- **排序规则**: `utf8mb4_0900_ai_ci`（不区分大小写、不区分重音）
- 建表语句中统一使用：`DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_0900_ai_ci`
- 连接串中指定：`characterEncoding=utf8mb4`、`connectionCollation=utf8mb4_0900_ai_ci`

## 表结构概览

| 表名 | 说明 |
|------|------|
| user | 用户表 |
| category | 商品分类表 |
| product | 商品表 |
| cart_item | 购物车项表 |
| order_main | 订单主表 |
| order_item | 订单明细表 |
| review | 商品评价表 |

## ER 关系简述

- **user**：独立实体，与 cart_item、order_main、review 为一对多。
- **category**：树形（parent_id），与 product 为一对多。
- **product**：属于 category；与 cart_item、order_item、review 为一对多。
- **cart_item**：关联 user、product（唯一约束 user_id + product_id）。
- **order_main**：关联 user；与 order_item 为一对多。
- **order_item**：关联 order_main、product（冗余商品快照）。
- **review**：关联 user、product、order_main（一个订单商品一条评价）。
