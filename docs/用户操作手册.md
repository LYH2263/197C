# 电商购物平台 - 用户操作手册

## 1. 环境与访问

### 1.1 启动系统

1. 确保已安装并启动 **Docker Desktop**。
2. 在项目根目录执行：
   ```bash
   docker compose up --build
   ```
3. 等待所有容器就绪（数据库初始化、后端启动、前端构建并挂载完成）。

### 1.2 访问地址

- **前端页面**：http://localhost:3197  
- **后端 API**：http://localhost:8197/api  

### 1.3 测试账号

| 管理员   | admin   | 123456 |
| 普通用户 | user1   | 123456 |
| 普通用户 | user2   | 123456 |

## 2. 普通用户操作

### 2.1 注册

1. 打开前端页面 http://localhost:3197 ，点击右上角或导航中的 **「注册」**。
2. 填写 **用户名**（必填，唯一）、**密码**（必填）、**昵称**、**手机**、**邮箱**。
3. 点击 **「注册」** 按钮。
4. 注册成功后会自动登录并跳转至首页；若提示「用户名已存在」请更换用户名。

### 2.2 登录

1. 点击 **「登录」**。
2. 输入 **用户名** 和 **密码**，点击 **「登录」**。
3. 登录成功后跳转至首页，右上角显示当前用户名；若账号被禁用会提示「账号已被禁用」。

### 2.3 浏览商品

1. **首页**：展示商品列表，可点击分类筛选。
2. **商品列表**：通过顶部导航进入「商品」或「分类」，查看分类下的商品。
3. **商品详情**：点击某商品进入详情页，可查看名称、价格、库存、主图、详情描述。
4. **搜索**：在商品列表或搜索框输入关键词，按商品名称搜索。

### 2.4 购物车

1. **加入购物车**：在商品详情页选择数量，点击 **「加入购物车」**；若未登录会提示先登录。
2. **查看购物车**：点击顶部导航 **「购物车」**，查看当前勾选与未勾选项、单价、数量、小计。
3. **修改数量**：在购物车列表中修改数量输入框，数量不得超过库存。
4. **勾选/取消勾选**：勾选需要结算的商品；仅勾选项参与结算。
5. **删除**：点击某商品的删除按钮可从购物车移除。
6. **结算**：勾选至少一件商品后，点击 **「去结算」** 进入订单确认页。

### 2.5 下单与支付

1. **填写收货信息**：在结算页填写 **收货人姓名**、**手机号**、**收货地址**（必填）。
2. **提交订单**：点击 **「提交订单」**，系统根据勾选购物车项生成订单；若库存不足会提示。
3. **支付（模拟）**：提交成功后进入订单详情页，点击 **「去支付」** 完成模拟支付，订单状态变为「已付款」。
4. **订单列表**：通过「我的订单」或导航进入订单列表，可查看全部订单及状态。
5. **订单详情**：点击某订单可查看订单主信息及商品明细。
6. **取消订单**：仅 **待付款** 订单可取消，在订单详情或列表中点 **「取消订单」**。

### 2.6 评价

1. **发表评价**：在订单详情或商品详情页，对已购商品进行 **评分（1–5 星）** 和 **文字评价**（可选），提交后即显示在商品评价列表。
2. **我的评价**：在「我的评价」页面查看当前用户发表的全部评价。

### 2.7 退出登录

点击右上角用户名或「退出」按钮，清除登录状态并返回首页。

## 3. 管理员操作

管理员除拥有普通用户全部功能外，还可进行以下操作。

### 3.1 用户管理

1. 使用管理员账号（如 admin）登录。
2. 通过导航进入 **「用户管理」** 或 **「管理员」→「用户管理」**。
3. **用户列表**：查看所有用户的用户名、昵称、状态、角色等。
4. **编辑用户**：可修改某用户的 **状态**（启用/禁用）、**角色**（普通用户/管理员）；禁用后该用户无法登录。
5. 保存后列表刷新，变更生效。

### 3.2 评价管理

1. 通过导航进入 **「评价管理」** 或 **「管理员」→「评价管理」**。
2. **评价列表**：查看全部用户对全部商品的评价（用户、商品、评分、内容、时间等）。
3. **删除评价**：对违规或不当评价点击 **「删除」**，确认后从系统中移除。

## 4. 常见问题

| 现象 | 可能原因 | 处理方式 |
|------|----------|----------|
| 无法登录 | 用户名或密码错误 | 检查输入或使用测试账号 user1/123456 |
| 无法登录 | 账号被禁用 | 联系管理员启用账号 |
| 加购/下单失败 | 未登录 | 先登录后再操作 |
| 加购/下单失败 | 库存不足 | 减少数量或选择其他商品 |
| 无法结算 | 未勾选购物车商品 | 在购物车中勾选要购买的商品后再点「去结算」 |
| 无法取消订单 | 订单已支付或已取消 | 仅「待付款」订单可取消 |
| 前端打不开 | 容器未就绪 | 等待 docker compose 完成后再访问 3197 端口 |
| 接口 401 | Token 过期或未登录 | 重新登录获取新 Token |

## 5. 界面与导航说明

- **顶部导航**：首页、商品/分类、购物车、我的订单、我的评价；右侧为登录/注册或当前用户名与退出。
- **管理员**：登录后可见「用户管理」「评价管理」等入口（具体以实际菜单为准）。
- **统一风格**：采用 Element Plus 组件，表单校验与提示以页面实际为准。

如有与当前版本界面不一致之处，以实际运行系统为准；功能逻辑与本文档一致。
