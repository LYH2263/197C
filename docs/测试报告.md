系统测试通过手工操作完成，覆盖主要业务流程，见下表。

| 场景 | 步骤简述 | 预期结果 |

## 3. 测试执行说明

### 3.1 运行单元测试与集成测试

在项目后端目录执行：

```bash
cd backend
mvn test
```

或仅运行指定测试类：

```bash
mvn test -Dtest=AuthServiceTest,CartServiceTest,OrderServiceTest,AuthControllerIntegrationTest
```

### 3.2 测试配置

- 测试使用 `application-test.yml` 激活（`@ActiveProfiles("test")"`）。
- 数据源为 H2 内存库（`jdbc:h2:mem:shop;MODE=MySQL`）。
- 建表脚本为 `src/test/resources/schema-h2.sql`，由 Spring Boot 在测试启动时自动执行（`spring.sql.init.mode=always`）。
